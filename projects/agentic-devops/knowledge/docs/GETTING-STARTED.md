# 快速开始

## 项目概览

Agentic DevOps 是基于 AgenticOS 理念的 AI 驱动 DevOps 新模式探索项目。

## 核心概念

### Agent-First
以 Agent 为核心，实现 DevOps 全链路智能化：
- **Dev Agent**：代码生成、审查
- **Test Agent**：智能测试
- **Deploy Agent**：自动部署
- **Ops Agent**：监控、自愈

### 关键优势
1. 自主决策，减少人工干预
2. 多 Agent 协作，提升效率
3. 持续学习，不断优化
4. 全链路可观测
